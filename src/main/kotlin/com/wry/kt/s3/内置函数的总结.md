# 内置函数

## apply
### str.apply
- apply函数返回类型，用于都是本身
- apply函数的匿名函数里边持有的是this==本身

## let
### str.let
- let函数返回类型，是根据匿名函数最后一行的变化而变化
- let函数的匿名函数里边持有的是it==集合本身

## run
